# Pandoc

Pandoc is the swiss-army knife of markup format conversions.

Download Pandoc from [here](https://github.com/jgm/pandoc/releases/).
Then install as you usually would.

## Verify Your Install

```{bash}
pandoc --version
```

yields:

